printf general format

 The printf function is used to display output in C. The simplest format is to simply display text:

printf("text to display here")

To display variables, you insert %code into the text string at the point you want the variable value to appear, where 'code' is a single character that indicates the type of variable that will be inserted (see the table below). You then add each variable as an additional argument to the printf function; there must be a 1 to 1 correspondence between %code appearances in the text string and extra arguments after the text string in the printf function. So for example, if x and y are the integers 2 and 3, the code

printf("x has the value %d and y=%d", x, y)

would display "x has the value 2 and y=3". printf does not add a newline character by default, if you want the next print to appear on the following line you must add the special character "\n" to the text argument of printf. In fact you can add "\n" anywhere in the text argument to insert a newline. The code

printf("x = %d \n y = %d \n", x, y)

would display:
x=2
y=3

When displaying decimal values (floats or doubles), I highly recommend using the code %g, as the compiler will automatically choose the best format & precision with which to display each value. For example, if x=1.246 and y=2.5:

printf("x=%g, y=%g\n", x, y)

will display "x=1.246, y=2.5". However, the code

printf("x=%f, y=%f\n", x, y)

will display the much less nice "x=1.246000, y=2.500000".

Table of conversion characters
Character Meaning
d, i Display an int in signed decimal notation.
o Display an int in unsigned octal notation (without a leading 0).
u Display an int in unsigned decimal notation.
x, X Display an int in unsigned hexadecimal notation (without a leading 0x or 0X). x gives lower case output, X upper case.
c Display a single char (after conversion to unsigned int).
e, E Display a double or float (after conversion to double) in scientific notation. e gives lower case output, E upper case.
f Display a double or float (after conversion to double) in decimal notation.
g, G g is either e or f, chosen automatically depending on the size of the value and the precision specified. G is similar, but is either E or f.
n Nothing is displayed. The corresponding argument must be a pointer to an int variable. The number of characters converted so far is assigned to this variable.
s Display a string. The argument is a pointer to char. Characters are displayed until a '\0' is encountered, or until the number of characters indicated by the precision have been displayed. (The terminating '\0' is not output.)
p Display a pointer (to any type). The representation is implementation dependent.
% Display the % character.
